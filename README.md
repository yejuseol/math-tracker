# Math Tracker

A web app I built to manage and track unit test results for my private math tutoring students.

Live: https://yejuseol.github.io/math-tracker/

## Features
- Unit test score logging with MCQ and FRQ breakdown
- Chapter-level achievement tracking and score trend visualization
- Role-based access for teacher, student, and parent accounts
- Covers Algebra 1/2, Precalculus, AP Precalculus, AP Calculus AB/BC

## Background
Started tracking student progress manually in Google Docs, then built this
to have a cleaner way to log scores, spot weak chapters, and share progress
with students and parents directly.

## Stack
HTML, CSS, JavaScript
