## 앱 개요

**앱 이름:** Chloe T Math — Unit Test Tracker
**URL:** yejuseol.github.io/math-tracker/
**목적:** 수학 과외 학생들의 단원 테스트 점수를 기록하고, 성취도에 따라 다음 학습 방향을 안내하는 웹 도구
**기술:** Firebase Auth + Firestore 기반 실시간 웹앱 (로그인 필요)

---

## 역할 (Role) 시스템

총 3가지 역할이 있으며, 역할에 따라 접근 가능한 화면이 다름.

### 선생님 (Teacher)
- 모든 학생 데이터 열람 및 수정 가능
- 점수 입력, 학생 관리, 통계 전체 접근 가능
- 별도 회원가입 불가 (기존 계정만 사용)

### 학생 (Student)
- 본인 점수 및 성취도만 열람 가능
- 다른 학생 데이터 접근 불가
- 점수 입력·학생 관리 탭 비노출
- 회원가입 시 자동으로 선생님 계정에 등록됨

### 학부모 (Parent)
- 자녀 이메일 연결을 통해 해당 자녀의 데이터만 열람 가능
- 학생 역할과 동일한 제한 적용

---

## 회원가입 / 로그인

### 로그인
- 이메일 + 비밀번호 입력 후 로그인
- 잘못된 정보 입력 시 한국어 오류 메시지 표시

### 회원가입 (학생)
입력 항목:
1. 이름
2. 이메일
3. 비밀번호 (6자 이상)
4. 역할: 학생 (Student) — 기본값
5. 학교 이름 (선택)
6. 학년: G8 / G9 / G10 / G11 / G12 / College 중 선택

가입 완료 시:
- Firebase Auth 계정 생성
- 선생님 화면의 학생 관리 탭에 자동 등록

### 회원가입 (학부모)
입력 항목:
1. 이름
2. 이메일
3. 비밀번호
4. 역할: 학부모 (Parent) 선택
5. 자녀 이메일 (자녀가 먼저 학생으로 가입한 이메일 입력)

## 화면 구성

로그인 후 화면은 좌측 사이드바 + 우측 메인 영역으로 구성.

### 사이드바
- 브랜드: Chloe T Math / Unit Test Tracker
- 메뉴: 대시보드 / 점수 입력 / 학생 관리 / 통계
  - 점수 입력 · 학생 관리는 선생님만 노출
- 계정 정보: 이름, 역할(선생님/학생/학부모), 로그아웃 버튼
- 뷰 모드 토글 (선생님 전용): 선생님 뷰 ↔ 학생/학부모 뷰 미리보기 전환

---

## 대시보드

### 선생님 뷰
- 요약 카드 4개: 전체 학생 수 / 총 테스트 횟수 / 평균 성취도 / 76% 이상 비율
- 최근 테스트 기록 테이블 (최근 10건): 학생명 / 과목+챕터 / 날짜 / 점수 / 성취도 배지 / 다음 스텝

### 학생·학부모 뷰
- 본인(또는 자녀) 데이터만 표시
- 동일한 요약 카드 + 최근 기록 테이블 (본인 기록만)

---

## 점수 입력 (선생님 전용)

두 가지 입력 모드 탭으로 구성:

### 탭 1: 총점 입력
입력 순서:
1. 학생 선택 (드롭다운)
2. 과목 선택 — 해당 학생의 수강 과목 목록으로 자동 필터링
3. 챕터 선택 — 선택 과목에 맞는 챕터 목록으로 자동 업데이트
4. 테스트 날짜
5. 만점 입력
6. 획득 점수 입력 → 성취도(%) 실시간 자동 계산 표시
7. 메모 (선택)

### 탭 2: 문항별 정오 입력
입력 순서:
1. 학생 선택
2. 과목 선택
3. 챕터 선택 → MCQ / FRQ 그리드 자동 생성
4. 날짜
5. MCQ (객관식): 문항별 ○/✕ 선택, 각 2점
6. FRQ (서술형): 문항별 획득 점수 직접 입력, 파트별 배점 표시
7. 자동 계산 점수 및 성취도 실시간 표시
8. 메모 (선택)

### 저장 후 결과 카드
저장 성공 시 결과 카드 표시:
- 성취도(%)와 학생 이름·챕터 표시
- 성취도 구간별 색상 (녹색/주황/빨강)
- 다음 스텝 안내 텍스트
- 자기주도 과제 안내 (56~75% 및 55% 이하 구간만)

### 성취도 기준

| 구간 | 색상 | 다음 스텝 |
|------|------|-----------|
| 76% 이상 | 녹색 | 다음 챕터 바로 진행 |
| 56–75% | 주황 | 다음 챕터 + 취약 단원 병행 보충 |
| 55% 이하 | 빨강 | 기존 챕터 복습 후 다음 챕터 시작 |

## 과목 및 챕터 구성

총 6개 과목, 각 과목별 챕터 목록:

### Algebra 1
Topic 1–15 (Mathematical Expressions → Statistics)

### Algebra 2
Ch.1–11 (Functions and Linear → Basic Statistics)

### Precalculus
Ch.1–11 (Functions → Sequence and Series)

### AP Precalculus
Ch.1–11 (Functions → Matrices)
※ Ch.8–11은 AP 시험 범위 외 표시

### AP Calculus AB
Ch.1–17 (Limit → Differential Equation)
Part 1: Differentiation (Ch.1–10)
Part 2: Integration (Ch.11–17)

### AP Calculus BC
Ch.1–24 (AB 전체 + BC 전용 Ch.18–24)
Part 3 BC 전용: Euler Method, Integration for BC, Infinite Series, Power Series, Taylor Series, Parametric Equation, Polar Equation

### MCQ / FRQ 자동 생성
- 챕터 선택 시 해당 챕터 소단원 수에 맞춰 MCQ/FRQ 자동 생성
- MCQ: 소단원당 1문항, 각 2점
- FRQ: 3–5문항, 총 배점 ≈ 전체의 60%
- Algebra 2 Ch.1은 상세 구성 고정 (Q1–Q14)

## 학생 관리 (선생님 전용)

### 화면 구성
- 과목 필터: 특정 과목 수강 학생만 필터링
- 학생 카드 목록: 회원가입한 학생 전체 표시

### 학생 카드 정보
각 카드에 표시되는 정보:
- 이름 + 학년
- 학교
- 연결 이메일
- 수강 과목 배지
- 테스트 횟수 / 평균 성취도 / 최근 성취도
- 최근 5건 기록 (과목 + 챕터 + 날짜 + 성취도 배지)

### 학생 정보 편집 (✏️ 버튼)
카드의 ✏️ 버튼 클릭 시 인라인 편집 폼 열림:
- 학교 수정
- 학년 변경
- 수강 과목 체크박스 선택 (복수 선택 가능)
- 저장 / 취소 버튼

### 학생 삭제
- ✕ 버튼 클릭 → 확인 후 해당 학생 + 모든 점수 기록 일괄 삭제

---

## 통계 (선생님·학생·학부모 공통)

### 필터
- 학생 필터: 전체 학생 / 특정 학생 선택
- 과목 필터: 전체 과목 / 특정 과목 선택

### 차트 3종
1. **챕터별 평균 성취도** (막대 그래프): 챕터별 평균 성취도(%) 비교
2. **성취도 분포** (파이 차트): 76%+ / 56–75% / ~55% 구간 비율
3. **점수 추이** (선 그래프): 시간순 성취도 변화, 동일 날짜면 입력 순서 기준

### 전체 기록 목록 테이블
- 학생명 / 과목+챕터 / 날짜 / 점수 / 성취도 / 다음 스텝 / 삭제 버튼
- 최신순 정렬

---

## 기타 동작 사항

- 실시간 업데이트: Firestore onSnapshot 기반, 다른 탭/기기에서 입력해도 즉시 반영
- 자동 오늘 날짜: 점수 입력 시 날짜 필드에 오늘 날짜 자동 입력
- 학생 연동: 학생이 회원가입하면 선생님 화면에 자동 등록 (이메일 연결 기준)
- 학부모 연동: 자녀 이메일 입력 시 해당 학생 데이터만 열람 가능
